An unofficial wrapper for the official **Hunter.FM** API, dependency-free.

Installation
------------

**Node.js 13.0.0 or older is required.**  

```sh-session
npm install DollnKey/hunter.fm
yarn add DollnKey/hunter.fm
pnpm add DollnKey/hunter.fm
```
